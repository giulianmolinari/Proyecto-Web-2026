# Proyecto Web 2026 - P_Web 1

Este proyecto corresponde a una práctica de desarrollo web, donde se implementa un sitio con varias páginas estáticas, estilos personalizados y recursos gráficos.

## Contenido de la carpeta

- **index.html** → Página principal del sitio.
- **about-us.html** → Sección de información sobre la organización/proyecto.
- **collection.html** → Página de colección o galería.
- **contact_us.html** → Formulario o sección de contacto.
- **eventos.html** → Página dedicada a eventos.
- **style.css** → Hoja de estilos principal.
- **logo-berserk.png** y **fondo-berserk.png** → Recursos gráficos del sitio.
- **tatuaje1.jpg ... tatuaje8.jpg, tatto 9.jpg** → Imágenes utilizadas en la galería/colección.

## Requisitos

- Navegador web moderno (Chrome, Firefox, Edge).
- No requiere instalación de dependencias externas, ya que es un proyecto estático.
